The queries.json.bz2 file contains list of ES queries that has been randomily generated 
from the AOL query dataset published in 2006. Only specific queries have been selected
and the rest of the file contains dummy queries. Only the query attribute is copied from
the AOL query dataset, the rest of the attributes are not in this file.

AOL's original README: http://www.gregsadetsky.com/aol-data/U500k_README.txt
